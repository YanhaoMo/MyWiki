# 目录结构
假设我们现在有一个名字叫做 foo 的项目，那么 foo 一定是这个项目的顶层目录。
在 foo 目录下，一般需要包含这些文件或目录：

- README 
- src
- include
- build
- test
- Makefile

以上是必须包含的，顺便，下面是一些可选项目：

- LICENSE
- doc

# 错误处理
